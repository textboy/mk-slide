猜数字游戏

guess.py
import random

secret = random.randint(1, 100)
attempts = 0

while True:
    guess = int(input("Guess: "))
    attempts += 1
    if guess == secret:
        print(f"Got it in {attempts}!")
        break
    print("Higher" if guess < secret else "Lower")

常见错误 / Mistakes

  • 1 IndentationError — 缩进错误，别混 Tab/空格
  • 2 = vs == — 赋值 vs 比较
  • 3 Off-by-one — range(5) = 0,1,2,3,4
  • 4 Type error — "5"+3 报错，要 int("5")+3
  • 5 Missing : — if/for/def 末尾要加冒号
