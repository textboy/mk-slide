## 3. CSS Variable Architecture

Every style defines these CSS custom properties:

```css
:root {
    /* Colors */
    --bg-primary: ...;
    --bg-secondary: ...;
    --text-primary: ...;
    --text-secondary: ...;
    --text-muted: ...;
    --accent: ...;
    --accent-secondary: ...;
    --border-subtle: ...;
    --surface: ...;

    /* Typography */
    --font-display: ...;
    --font-body: ...;
    --font-mono: ...;

    /* Sizes (ALL use clamp()) */
    --title-size: clamp(2.8rem, 6.5vw, 5.5rem);
    --h2-size: clamp(1.8rem, 3.8vw, 3.2rem);
    --h3-size: clamp(1.2rem, 2vw, 1.6rem);
    --body-size: clamp(1rem, 1.6vw, 1.25rem);
    --small-size: clamp(0.8rem, 1.2vw, 1rem);
    --stat-size: clamp(3.5rem, 8vw, 6.5rem);

    /* Spacing (ALL use clamp()) */
    --slide-padding: clamp(1rem, 4vw, 4rem);
    --content-gap: clamp(0.5rem, 2vw, 2rem);
    --element-gap: clamp(0.25rem, 1vw, 1rem);

    /* Animation */
    --ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
    --duration-normal: 0.6s;
}
```

## 4. Layout DNA

Each style defines a "Layout DNA" — the structural patterns that make it unique:

- **Slide mechanism**: typically `position: fixed` with opacity transition
- **Background treatment**: solid color, gradient, texture, pattern
- **Title alignment**: centered, left-aligned, asymmetric, lower-third
- **Navigation style**: dots, tabs, breadcrumbs, thumbnails
- **Component structure**: cards, full-bleed, split-panel, columns
- **Animation approach**: fade, slide, clip-path, scale, glitch

## 5. Font Strategy

- **Display fonts**: Distinctive, characterful — Google Fonts or Fontshare
- **Body fonts**: Readable, versatile — paired with display fonts
- **Mono fonts**: For technical content (optional, per style)
- **CJK fonts**: Always included when Chinese/Japanese text is present
- **Never system fonts alone** — always include web fonts
- **Fallback stacks**: Always comprehensive, CJK system fonts for China

### 5.1 China Network Fallback

Google Fonts may be slow/blocked in mainland China. Always include:
- System CJK fonts: `'PingFang SC', 'Microsoft YaHei', sans-serif`
- Consider `fonts.googleapis.cn` mirror
- Consider embedding critical font subsets as base64
