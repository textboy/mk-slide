Lessons Learned

Five things we'd do differently.

Schema Registry from day one
Schema evolution broke us three times before we added Avro validation.
Start with Flink SQL, not Java
80% of our processing logic fits in SQL. Java is for the remaining edge cases.
Monitor consumer lag religiously
Lag alerts prevented more incidents than any code review.
Backpressure is a feature
Instrument it, tune it, respect it. Don't fight the system.
Exactly-once is a spectrum
Know your actual consistency needs before over-engineering.
